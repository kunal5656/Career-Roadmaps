    Web
    Development


    Web Development is a technology that focuses on development of websites that users surfs all the time on the internet and Web Developers are those that focuses on development of these Websites.Web Development consists of both the Front-End as well as Back-End.Web Developers can either be Front-End developers or Back-End developers. Web development can range from developing a simple single static page of plain text to complex web applications, electronic businesses, and social network services. Web development has been a growing industry. The growth of this industry is being driven by businesses wishing to use their Web site to advertise and sell products and services to customers.With the growing set of tools and Technologies helped developers to build much more rich and dynamic websites with greater functionalities.


    Web Development can be divided into:

    Front-End Web Development

    Front-End web development is about developing rich user interfaces,in short it is all about developing the Graphical User interface (GUI) of the Website that allows the users to view and interact with the website.

    Back-End Web Development

    Back-End Development is about developing the backend functionality to the website.Its stores and arranges the data.It is the part of the website which cannot be interacted or handeled by the end user.
